Project Biography

This project focused on building a k-nearest neighbours classifier in Python to predict Titanic passenger survival. The work involved preprocessing raw data, handling missing values, scaling features appropriately, and selecting a model setup that could produce strong classification performance.

A key part of the project was understanding that machine learning performance depends heavily on data preparation. Rather than only training the model, I worked through the steps that make the data usable and comparable, then evaluated results using metrics such as accuracy and the confusion matrix.

This project helped me strengthen my programming and analytical skills while also giving me practical experience with the machine learning workflow from raw dataset to final evaluation. It showed how engineering thinking can also apply strongly in data-focused problem solving.
